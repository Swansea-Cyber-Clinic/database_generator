Welcome to database_generator's documentation!
==============================================

**database_generator** is a Python project for generating a SQLite database for Swansea Cyber Clinic's `Interactive Service Mapping <https://cyberclinic.wales/service_mapping>`_.

.. note:: 
  This project is under active development. If you would like to contribute to development, `let us know <mailto:cyberclinic@swansea.ac.uk>`_!
